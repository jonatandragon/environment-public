A light color theme for IDEs, designed around C# with ReSharper. Available for:

* [Rider](https://plugins.jetbrains.com/plugin/28585-pulsar-light-theme) - JetBrains Marketplace
* [Visual Studio](https://marketplace.visualstudio.com/items?itemName=JD-9cbb2fe7-f10e-4487-ac01-ce7e138ec3f2.pulsar-light-theme) - Visual Studio Marketplace (C# with ReSharper only)

See [`ide/pulsar`](ide/pulsar) for color details.
