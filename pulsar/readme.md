## Colors

Color categories are defined for C#. Other languages may use them differently.

### Foregrounds

| Category                               | Example                                  | Color     |
|----------------------------------------|------------------------------------------|-----------|
| Keyword                                | `var`, `int`, `public`, `static`         | `#EA62CA` |
| Keyword 2 (flow)                       | `for`, `switch`, `if`                    | `#B601EA` |
| Interface                              | `IEnumerable`                            | `#37A925` |
| Class                                  | `Program`                                | `#1075E4` |
| Struct                                 | `TimeSpan`                               | `#2C91AF` |
| Enum type                              | `DayOfWeek`                              | `#FF6D12` |
| Enum member                            | `Monday`                                 | `#FFA064` |
| Method                                 | `CreateBuilder()`                        | `#00B0CF` |
| Identifier (var, namespace, param)     | `myVariable`, `MyApp.Domain`             | `#000000` |
| String                                 | `"An error occured."`                    | `#E6A100` |
| String format item                     | `{itemA}`                                | `#FF6D12` |
| Comment                                | `// Ignore.`, `/// <summary>`            | `#BCBCBC` |
| Comment TODO                           | `// TODO Fix this.`                      | `#008DD9` |
| Preprocessor                           | `#if #endif`                             | `#A9A9A9` |
